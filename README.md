# Notes App

Aplikacja webowa umożliwiająca rejestrację użytkowników, logowanie oraz zarządzanie własnymi notatkami (CRUD).

## Wersja online

Frontend:  
https://notes-app-three-pi.vercel.app/login.html  

Backend API:  
https://notes-app-6p5k.onrender.com  

---

## Wymagania

- Node.js
- npm

---

## Instalacja

1. Sklonuj repozytorium:

git clone https://github.com/TWOJ_LOGIN/notes-app.git

2. Przejdź do folderu backend:

cd notes-app/backend

3. Zainstaluj zależności:

npm install

---

## Uruchomienie

W folderze backend:

node server.js

Serwer uruchomi się pod adresem:  
http://localhost:3000

Frontend uruchom przez otwarcie pliku:

frontend/login.html

w przeglądarce lub przez Live Server.
