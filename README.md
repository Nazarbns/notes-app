# Notes App

Aplikacja webowa umożliwiająca rejestrację użytkowników, logowanie oraz zarządzanie własnymi notatkami (CRUD).

## Wersja online

Frontend:  
https://notes-app-three-pi.vercel.app/login.html  

Backend API:  
https://notes-app-6p5k.onrender.com  

---

## Wymagania

- Node.js
- npm

---

## Instalacja lokalna

1. Sklonuj repozytorium:

git clone https://github.com/nazarbns/notes-app.git

2. Przejdź do folderu projektu:

cd notes-app/backend

3. Zainstaluj zależności:

npm install

---

## Uruchomienie lokalne

W folderze `backend` uruchom:

node server.js

Serwer będzie dostępny pod adresem:

http://localhost:3000

Frontend uruchom poprzez otwarcie pliku:

frontend/login.html
